# Task 0.4: CodeBuild Project

## Description

Define the shared CodeBuild project used by the Build Lambda to run builds/tests.

## Requirements

- Project named `{stackTag}-dakoder-build`
- Source type: S3 (bucket and path provided at build start time)
- Environment: standard Node.js managed image (aws/codebuild/standard)
- Default buildspec: `npm install && npm test` (overridden if repo contains `buildspec.yml`)
- Build Lambda has permission to start builds on this project
- S3 bucket granted as source and artifact location
- `BUCKET_NAME` and `STACK_TAG` passed as environment variables

## Inputs

- Stack tag
- S3 bucket reference
- Build Lambda function reference (for permissions)

## Outputs

- CodeBuild project resource
- Project name exported as environment variable on Build Lambda (`CODEBUILD_PROJECT_NAME`)

## Dependencies

- 0.1
- 0.2
- 0.3

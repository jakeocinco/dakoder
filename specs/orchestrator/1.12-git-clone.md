# Task 1.12: Git Clone into Workspace

## Description

Clone the target repo into the S3 workspace on new task creation.

## Requirements

- Uses `REPO_URL` and `DEFAULT_BRANCH` from environment variables (set by CDK from `.dakoder.toml`)
- Fetches git token from Secrets Manager using `GIT_CREDENTIALS_SECRET` env var
- Injects token into clone URL as `https://x-access-token:{token}@github.com/...`
- Clones with `--depth 1` to the specified branch into `/tmp/{taskId}`
- Uploads cloned files (excluding `.git/`) to `s3://{bucket}/{taskId}/`
- Handles clone failures (bad URL, auth, network) gracefully — reports as task failure

## Dependencies

- 1.4

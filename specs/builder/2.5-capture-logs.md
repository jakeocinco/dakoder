# Task 2.5: Capture Build Logs

## Description

Retrieve build output from CloudWatch Logs after build completes.

## Requirements

- Gets log group/stream from the build result
- Fetches log events from CloudWatch Logs
- Concatenates into a single string
- Truncates if over a size limit (e.g., 50KB) to avoid payload bloat
- Writes full logs to `s3://{bucket}/{taskId}/builds/{timestamp}.log`

## Dependencies

- 2.4

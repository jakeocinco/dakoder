# Task 2.4: EventBridge Rule (CDK)

## Description

CDK construct that routes CodeBuild completion events to the Build Lambda.

## Requirements

- EventBridge rule matching:
  - `source: "aws.codebuild"`
  - `detail-type: "CodeBuild Build State Change"`
  - `detail.build-status`: `SUCCEEDED`, `FAILED`, or `STOPPED`
  - `detail.project-name`: the shared CodeBuild project name
- Target: Build Lambda function
- Grant EventBridge permission to invoke the Build Lambda

## Dependencies

- 0.3 (Lambda functions defined in CDK)
- CDK CodeBuild project definition

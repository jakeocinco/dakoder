# Task 1.15: Prompt Tracking

## Description

Write each constructed prompt to the task's S3 workspace for auditability and debugging.

## Requirements

- On every prompt build, write the prompt to `s3://{bucket}/{taskId}/prompts/{iteration}.md`
- Iteration 0 is the initial prompt; subsequent numbers are retries
- Stored as plain text (markdown)
- Does not block the main flow on write failure (best-effort)

## Dependencies

- 1.4, 1.5

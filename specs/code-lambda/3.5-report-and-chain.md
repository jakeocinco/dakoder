# Task 3.5: Report and Chain

## Description

After agent completion, invoke the next Lambda in the chain.

## Requirements

- On success: async invoke Build Lambda with `{ type: "start", taskId }`
- On failure: async invoke Orchestrator with `{ type: "callback", taskId, status: "build-failed", buildLogs: "<error>" }`
- Uses `BUILD_FUNCTION_NAME` and `ORCHESTRATOR_FUNCTION_NAME` env vars
- Handles invoke errors gracefully (logs, does not crash)

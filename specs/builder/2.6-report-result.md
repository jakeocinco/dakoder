# Task 2.6: Report Result

## Description

Async invoke the next Lambda based on build outcome.

## Requirements

- On success: invoke Review Lambda with `{ taskId }`
- On failure: invoke Orchestrator with `{ type: "callback", taskId, status: "build-failed", buildLogs }`
- Uses `REVIEW_FUNCTION_NAME` and `ORCHESTRATOR_FUNCTION_NAME` env vars
- Handles invoke errors gracefully (logs and does not crash)

- 2.5

# Task 1.6: Async Lambda Invocation

## Description

Utility to async invoke other Lambda functions (fire-and-forget).

## Requirements

- Uses AWS SDK `InvokeCommand` with `InvocationType: 'Event'`
- Accepts target function name and JSON payload
- Shared utility — usable by orchestrator and worker Lambdas
- Handles invoke errors (function not found, permission denied)
- Returns confirmation that invoke was accepted (not the result)

## Inputs

- Target Lambda function name
- Payload object

## Outputs

- `{ accepted: boolean, error?: string }`

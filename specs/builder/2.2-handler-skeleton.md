# Task 2.2: Handler Skeleton

## Description

Lambda handler that receives either a start event or an EventBridge build completion event.

## Requirements

- Exports a Lambda handler matching AWS Lambda signature
- Routes `{ type: "start", taskId }` → start build logic
- Routes `{ "detail-type": "CodeBuild Build State Change" }` → build completion logic
- Rejects malformed events with clear error

# Task 3.3: S3 Workspace Sync

## Description

Download the task's S3 workspace to `/tmp` and upload changes back after the agent completes.

## Requirements

- Downloads all files from `s3://{bucket}/{taskId}/` to `/tmp/{taskId}/`
- After agent completes, uploads modified/new files back to S3
- Detects changes via file modification times or hash comparison
- Skips `metadata.json` and `prompts/` directory on upload (orchestrator-owned)
- Handles large workspaces within `/tmp` 10GB limit

- 3.1

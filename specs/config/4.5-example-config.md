# Task 4.5: Example Config and Prompt Files

## Description

Create a working `.dakoder.toml` example in this repo and placeholder agent prompt files.

## Requirements

- `.dakoder.toml` at repo root with realistic values for the dakoder project itself
- `.dakoder/code-prompt.md` with a placeholder system prompt for the coding agent
- `.dakoder/review-prompt.md` with a placeholder system prompt for the review agent
- The example config must parse successfully with `parseConfig()`
- Add a test that loads and parses the actual `.dakoder.toml` from the repo root

# Task 1.3: Task ID Generation

## Description

Utility to generate unique task IDs for keying S3 workspaces.

## Requirements

- IDs are URL-safe and S3-key-safe (alphanumeric, hyphens, underscores only)
- Sufficient entropy to avoid collisions across concurrent tasks
- Uses task name as the base (slugified)
- Deterministic length

## Inputs

- Task name/descriptor string

## Outputs

- `string` — the generated task ID

# Task 2.3: Start CodeBuild

## Description

Start a CodeBuild build using the task's S3 workspace as source.

## Requirements

- Uses `StartBuildCommand` from `@aws-sdk/client-codebuild`
- Source location: `s3://{bucket}/{taskId}/`
- Uses `CODEBUILD_PROJECT_NAME` env var for the project name
- Passes `taskId` as an environment variable override to the build
- Returns the build ID for correlation
- On start failure: immediately reports build-failed to orchestrator

## Dependencies

- 2.1

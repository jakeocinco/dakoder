# Task 1.13: Git Push and PR Creation

## Description

On finalization, push the task branch and open a pull request.

## Requirements

- Downloads workspace from S3 to `/tmp/{taskId}`
- Initializes git repo, creates branch with `BRANCH_PREFIX` + taskId
- Commits all files, pushes to remote using `GIT_CREDENTIALS_SECRET` token
- Opens a PR via GitHub API targeting `DEFAULT_BRANCH`
- PR title derived from task description
- All env vars (`REPO_URL`, `DEFAULT_BRANCH`, `BRANCH_PREFIX`, `GIT_CREDENTIALS_SECRET`) set by CDK

## Dependencies

- 1.9, 1.12

# Task 1.12: Git Clone into Workspace

## Description

Clone the target repo into the S3 workspace on new task creation.

## Requirements

- Clones repo URL and branch into `/tmp` in the Lambda
- Uploads cloned files to the task's S3 workspace folder
- Handles clone failures (bad URL, auth, network) gracefully
- Uses git credentials from a secure source (Secrets Manager or SSM Parameter Store)

## Dependencies

- 1.4

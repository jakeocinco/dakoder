# Task 3.4: Agent Invocation

## Description

Run the Claude Agent SDK `query()` against the local workspace.

## Requirements

- Calls `query()` with prompt and options:
  - `cwd`: local workspace path
  - `allowedTools`: `["Read", "Write", "Edit", "Glob", "Grep"]`
  - `permissionMode`: `"bypassPermissions"`
  - `allowDangerouslySkipPermissions`: `true`
  - `maxTurns`: `25`
  - `settingSources`: `[]`
- Iterates the async generator to completion
- Extracts `SDKResultMessage` for success/failure status
- Handles `error_max_turns` and `error_during_execution` subtypes
- Logs agent progress (turn count, cost)

## Dependencies

- 3.1, 3.3

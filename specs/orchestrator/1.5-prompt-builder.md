# Task 1.5: Prompt Builder

## Description

Construct prompts for the Code Lambda based on task context and prior feedback.

## Requirements

- Builds initial prompt from task description and repo context
- On build failure retry: appends build logs/errors to prompt
- On review rejection retry: appends review comments to prompt
- Accumulates history of prior feedback across iterations
- Prompt output is a single string ready to pass to Code Lambda

## Inputs

- Task description
- Iteration number
- (Optional) Build logs from failed build
- (Optional) Review comments from rejected review

## Outputs

- `string` — constructed prompt

## Dependencies

- 1.1

# Task 1.11: Wire Up Handler

## Description

Connect the handler skeleton to the actual implementations (workspace creation, prompt building, Code Lambda invocation, callback routing).

## Requirements

- `handleNewTask` generates task ID, creates S3 workspace, builds initial prompt, invokes Code Lambda
- `handleCallback` routes to the callback handler (finalize, retry, or halt)
- Uses existing modules: `task-id`, `workspace`, `prompt`, `callback`, `finalize`

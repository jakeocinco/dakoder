# Task 4.2: Types and Defaults

## Description

Define TypeScript interfaces for the config schema, default values, and a custom error class.

## Requirements

- `DakoderConfig` interface with all sections as named types
- `ProjectConfig` interface for `[project]`
- `GitConfig` interface for `[git]`
- `WorkflowConfig` interface for `[workflow]`
- `TriggersConfig` interface for `[triggers]`
- `AgentConfig` interface for per-phase agent settings (`[agents.code]`, `[agents.review]`)
- `AgentsConfig` interface containing optional `code` and `review` fields typed as `AgentConfig`
- Constants for default values:
  - `default_branch`: `"main"`
  - `branch_prefix`: `"dakoder/"`
  - `max_iterations`: `5`
  - `agents.code.max_turns`: `25`
  - `agents.review.max_turns`: `10`
- `ConfigError` class extending `Error` with a descriptive message
- Valid trigger types as a const array: `["pr_comment", "merge"]`
- All types exported for use by orchestrator and code-lambda
